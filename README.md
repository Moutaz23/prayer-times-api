# Prayer Times API

Eine arabische Webanwendung zur Anzeige täglicher Gebetszeiten für verschiedene Städte über eine externe API.

## Funktionen

- Anzeige der täglichen Gebetszeiten
- Auswahl verschiedener Städte
- Automatische Aktualisierung von Stadt und Datum
- Arabische Benutzeroberfläche
- Responsive Darstellung

## Verwendete Technologien

- HTML
- CSS
- JavaScript
- Axios
- Aladhan API
- GitHub Pages

## Live-Demo

[Projekt öffnen](https://moutaz23.github.io/prayer-times-api/)

## Installation

1. Repository herunterladen oder klonen
2. `index.html` im Browser öffnen
3. Internetverbindung erforderlich, damit die API-Daten geladen werden können
